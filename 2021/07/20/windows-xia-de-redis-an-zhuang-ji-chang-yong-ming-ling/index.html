简 言

REmote DIctionary Server(Redis) 是一个由 Salvatore Sanfilippo 写的 key-value 存储系统，是跨平台的非关系型数据库。

Redis 是一个开源的使用 ANSI C 语言编写、遵守 BSD 协议、支持网络、可基于内存、分布式、可选持久性的键值对(Key-Value)存储数据库，并提供多种语言的 API。

Redis 通常被称为数据结构服务器，因为值（value）可以是字符串(String)、哈希(Hash)、列表(list)、集合(sets)和有序集合(sorted sets)等类型。

简言介绍及下方部分内容借鉴来自菜鸟教程，官方指路👉菜鸟教程–redis

1. redis安装

  首先找到redis的下载路径，官方指路👉redis下载，选择相应的版本进行下载，下载完成之后，一直按next，在最后默认安装路径是C盘文件夹，这里推荐在D盘建一个文件夹来安装，根据个人意愿选择安装路径。😀

 文件结构目录如下图所示👇

  • redis-server.exe： 服务端程序，提供 redis 服务
  • redis-cli.exe: 客户端程序，通过它连接 redis 服务并进行操作
  • redis.windows.conf： 配置文件，将 redis 作为普通软件使用的配置，命令行关闭则 redis 关闭

2. redis的使用

当安装好redis后，启动程序。
第一种方法：

  • Windows+R打开命令提示窗，输入cmd，默认是C盘，需要切换到redis安装目录，我的是D:\redis下
  • 之后执行redis-server.exe redis.windows.conf命令，演示一下如图👇

第二种方法：

  • 创建一个.txt文档，输入redis-server.exe redis.windows.conf保存 命名可以随便，建议start.txt
  • 把该文本文件后缀名改为.bat并放在redis根目录下
  • 点击start.bat文件启动
    出现上图情况，就启动成功了。

注意事项

  • 第一次启动redis-server.exe程序时，报错或者一闪而过，解决方法：启动redis-cli.exe客户端-执行关闭命令shutdown-退出客户端-执行服务端执行命令redis-server.exe redis.windows.conf或者start.bat即可

  • 在使用redis的过程中，服务端server一定要打开，不能×掉，不然是不能使用客户端的，出现的结果可能就是这样，如图1👇 ，正确操作如图2，可以把服务端窗口隐藏，留服务端使用，切不可关闭服务端。

先来了解一下redis的数据类型分：string（字符串），hash（哈希），list（列表），set（集合）及zset(sorted set：有序集合)。
配合redis命令来了解redis数据类型，下面开始
① String（字符串类型）
string 类型是 Redis 最基本的数据类型，string 类型的值最大能存储 512MB。

② Hash（哈希）
Redis hash 是一个键值(key=>value)对集合。Redis hash 是一个 string 类型的 field （字段）和 value（值） 的映射表，hash 特别适合用于存储对象。
样例：
hmset *** field1 "Hello" field2 "World"

  • ***类似于一个对象类
  • field是一个字段，类似于对象类的属性
  • 而”“内的内容，类似于对象类属性对应的值

③ List列表

Redis 列表是简单的字符串列表，按照插入顺序排序。你可以添加一个元素到列表的头部（左边）或者尾部（右边）。应用场景：关注列表、粉lpush丝列表等，最新消息排行、每篇文章的评论
