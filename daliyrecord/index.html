简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
可以通过Git进行文章的更改，发布，可以部署到任意的git平台（GitHub、Coding、Gitee……）来托管自己的项目。

2020-10-15

  • 熟悉hexo文件夹的组成，一般情况下，hexo根目录文件夹名如下：
    ├── _config.yml # Hexo的全局配置文件，在此配置网站的主要参数
    ├── package.json # npm软件包以及版本信息
    ├── scaffolds # 模版文件夹
    ├── source # 资源文件夹
    | ├── _drafts # 草稿/模板文件
    | └── _posts # 文章（每篇文章使用markdown格式写好保存在这里）
    └── themes # 主题文件夹
  • 在了解基本结构之后，安装依赖: 在hexo目录内打开Git Bash Here,使用NPM安装hexo必须的依赖：、npm install 、

2020-10-16~2020-10-22

  • 部署到GitHub，步骤如下：关联GitHub仓库，链接git到远程仓库，获得密钥，强制ssh，修改hexo主题配置等操作，基本配置完成后，就可以通过仓库名访问hexo原生博客了，通过GitHub上的项目buttfly主题,来部署博客。在调配期间博客崩了六七次的样子，当时由于不知道问题出在哪，以及对hexo命令集不熟练，导致我重装了3回左右，不过到后来熟悉之后容易上手多了，既是出错也能找到问题所在之处。
  • 为什么在其期间没做笔记，刚开始也没想到笔记，后来受到同学启发，才想到要做笔记，毕竟是对自己做的东西的总结和思考吧。这期间崩的次数有点多就省了不少叙述。

2020-10-23

  • 添加日志，修改标签页，分类页的顶部背景图片（top_img）
  • social: 解决社交图标（QQ）的跳转链接问题。
    解决方法如下：tencent://AddContact/?fromId=45&fromSubId=1&subcmd=all&uin=QQ号码
  • local_search: 添加搜索功能，添加此功能前需要下载相应插件，有两种搜索方式：Algolia搜索 和 本地搜索，本人下载的是本地搜索hexo-generator-search，为什么呢？因为方便，步骤简单。本地搜索搜索范围默认涵盖本博客所有帖子（说白了就是搜索范围包括文章标题和文章内容中和标题部分重合的文章、包括搜索关键词和文章内容有相同部分的文章）。
  • 下载命令：npm install hexo-generator-search --save，下载后把本地搜索false改为true就大功告成了，如果search功能没有实现尝试清除缓存（hexo clean）一下，基本就可以了。
  • comments: 添加评论功能,使用通用设置的配置。同search功能修改有类似。
